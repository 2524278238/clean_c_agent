# C盘空间清理与 AI 分析 Agent (clean_c_agent)

这是一个基于 PyQt6 和大语言模型（如 DeepSeek/OpenAI）构建的智能 Windows C 盘空间清理与分析工具。

## 🌟 核心功能

### 1. 扫描与清理
*   **安全扫描**：自动排除 `WeChat` 和 `QQ` 的核心数据库文件（`.db`, `.sqlite`, `.bakdb` 等），防止聊天记录误删。
*   **分类扫描**：
    *   **常规系统清理**：扫描 Windows 临时文件和系统更新缓存。
    *   **社交软件专清**：专项扫描微信、QQ 的图片、视频和接收文件目录。
    *   **用户文档**：细分 `C:\Users\<User>\Documents` 目录。
*   **仪表盘**：直观展示 C 盘当前总容量、已用空间和可用空间的动态可视化图表。
*   **文件操作**：支持将文件永久删除，或**安全移动到 D 盘**（作为备用回收站）。

### 2. 恢复中心 (D 盘)
*   **路径记忆**：移动到 D 盘的文件会记录其原始 C 盘路径。
*   **一键恢复**：随时可以在界面中选择已移动的文件，将其还原回 C 盘的原始位置。

### 3. AI 空间分析 (核心亮点)
*   **目录树分析**：提供类似 `SpaceSniffer` 的目录占用分析，支持按文件大小排序、缓存加速和增量扫描。
*   **AI 一键分析**：接入大模型（支持 DeepSeek 等），一键分析当前目录下最占用空间的文件夹，输出结构化的 Markdown 分析报告，告诉你每个文件夹的作用、风险等级和清理建议。
*   **AI 对话与自主探查**：
    *   内置 AI 对话助手，支持上下文记忆。
    *   **工具调用能力**：AI 可以自主调用本地文件系统工具（`list_directory`），帮你“看”文件夹里面到底有什么，并给出清理建议。

## 🚀 快速开始

### 方式一：直接运行 (推荐)
从 [Releases](#) 下载最新的 `C盘清理与AI分析Agent.exe`，双击即可运行（无需安装 Python 环境）。

### 方式二：源码运行
1. 克隆仓库：
   ```bash
   git clone https://github.com/2524278238/clean_c_agent.git
   cd clean_c_agent/cleaner_app
   ```
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python main.py
   ```

## ⚙️ AI 配置指南
为了使用 AI 分析和对话功能，你需要配置 API Key。
1. 打开软件，点击菜单栏的 **选项 -> 软件设置**。
2. 选择提供商（如 DeepSeek）。
3. 填入你的 `API Key`。
4. （可选）修改 `Base URL` 和 `Model` 名称。
5. 点击“保存”。

## 🛠️ 技术栈
*   **GUI 框架**: PyQt6
*   **AI 驱动**: OpenAI Python SDK (支持所有兼容 OpenAI 格式的 API)
*   **系统交互**: `os`, `shutil`, `psutil`, `concurrent.futures`
*   **UI 渲染**: Markdown, Pygments
